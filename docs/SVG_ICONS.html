📝 使用说明

  1. 复制每个SVG代码块
  2. 保存为对应的文件名（如 tab-home.svg）
  3. 使用在线工具或软件将SVG转换为PNG格式
  4. 建议生成32x32px和64x64px两个尺寸
  5. 将PNG文件放入 Wine_miniapp/images/ 目录
  6. 在 app.json 中恢复标签栏配置

🔗 推荐的SVG转PNG工具

  • 在线工具: convertio.co、cloudconvert.com
  • 设计软件: Figma、Adobe Illustrator
  • 浏览器工具: 直接在浏览器中截图这个页面的图标
  • 命令行工具: ImageMagick、Inkscape

提示: 每个图标都经过精心设计，符合调酒小程序的蓝紫色主题和现代简约风格。激活状态使用了玻璃拟态效果，与应用整体设计风格一致。
